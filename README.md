# CacheIt

A zero-trust, end-to-end encrypted note-taking project.

## Overview

CacheIt is a cross-platform note-taking application built with real-time sync,
per-field client-side encryption, and offline-first support. The server never
has access to plaintext note content. This repo holds the project's shared
documentation, architecture decisions, and design specs — the code lives in
the linked repos below.
